Projects

Automated Invoice Processor
python, ocr, json
Problem: A team was spending hours manually extracting data from hundreds of PDF invoices each month.
Solution: Developed a Python script that uses OCR to automatically read invoices, parse key details (like vendor, date, and amount), and output structured JSON, eliminating manual entry and human error.
Corporate Travel Request System
Ms power automate, sharepoint
Problem: An unstructured and chaotic travel request process was causing delays and confusion.
Solution: Designed and implemented a streamlined workflow using the Microsoft Power Platform. The system handles forms, multi-level approvals, and automated email updates, all tracked in a central SharePoint list.
Enterprise Email Parser
python, api integration
Problem: Important information from inbound emails was being missed or processed too slowly.
Solution: Built a service to monitor an enterprise inbox, automatically extract critical information based on predefined rules, and store the data in a central company application, saving hours of repetitive daily checks.

Problem: A shipping line needed a professional online presence to attract clients and handle inquiries.
Solution: Created a full-stack, responsive website with multiple pages and a contact form that captures leads and stores them in a MongoDB database for efficient follow-up by the sales team.
